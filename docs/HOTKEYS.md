# MU Online Client — Hotkey Reference

## Global UI

| Key | Action |
|-----|--------|
| `Esc` | Open System Menu |
| `Tab` | Toggle Minimap (Toggle Siege Warfare window in Battle Castle) |
| `M` | Toggle Move Map |
| `F1` | Toggle Help window |

## Character & Social

| Key | Action |
|-----|--------|
| `C` | Toggle Character Info |
| `A` | Toggle Master Level window (Master-class characters only) |
| `F` | Toggle Friends List (level 6+ only) |
| `P` | Toggle Party window |
| `G` | Toggle Guild Info |
| `D` | Toggle Command window (disabled in Chaos Castle / Strife maps) |
| `B` | Toggle Gens Ranking |
| `U` | Toggle Window Menu |

## Inventory

| Key | Action |
|-----|--------|
| `I` / `V` | Toggle Inventory |
| `K` | Toggle Inventory Extension bag |
| `L` | Toggle Repair mode |
| `S` | Toggle Personal Shop |
| `Ctrl + Q/W/E/R` | Assign hovered item to quick-use slot Q/W/E/R |

## Item Quick-Use

| Key | Action |
|-----|--------|
| `Q` | Use item assigned to slot Q |
| `W` | Use item assigned to slot W |
| `E` | Use item assigned to slot E |
| `R` | Use item assigned to slot R |
| `Space` | Auto-pick up nearby items |

## Skills & Pets

| Key | Action |
|-----|--------|
| `1` – `9`, `0` | Activate skill hotkey 1–9, 0 |
| `Ctrl + 1` – `0` | Assign skill to hotkey slot |
| `Shift + 1` – `4` | Cycle pet command (Default / Attack / Follow / Stay) |

## Mu Helper

| Key | Action |
|-----|--------|
| `Z` | Open / close Mu Helper configuration panel |
| `F8` | Start / stop (play / pause) Mu Helper |

In Mu Helper's Other Settings tab, friend, guild member, and party requests can
be set to `On` to show the normal confirmation window, `Off` to reject requests
automatically, or `Auto` to accept requests automatically.

## Chat

| Key | Action |
|-----|--------|
| `Enter` | Open chat input / send message |
| `F2` | Toggle Whisper / All Messages view |
| `F3` | Toggle Whisper Send mode |
| `F4` | Auto-size Chat Log window |
| `F5` | Toggle Chat Log frame visibility |
| `Page Up` | Scroll chat log up |
| `Page Down` | Scroll chat log down |
| `↑ / ↓` | Navigate chat history |

## NPC Shop

| Key | Action |
|-----|--------|
| `L` | Toggle Repair tab |
| `Shift + L` | Repair all items at once |

## Storage / Vault

| Key | Action |
|-----|--------|
| `H` | Toggle extended vault (if unlocked) |

## Shop

| Key | Action |
|-----|--------|
| `X` | Open / close In-Game Cash Shop |

## Camera

| Key | Action |
|-----|--------|
| `F9` | Cycle camera mode |
| `F10` | Toggle zoom lock |
| `F11` | Reset camera view |

## Mouse Shortcuts

| Input | Action |
|-------|--------|
| `Left Click` | Move / interact / use skill |
| `Right Click` | Move (alternate) / context actions |
| `Alt + Right Click` | Open Quick Command menu for a nearby player |
